[SVN 설치 및 사용법]	
	 - svn server 를 검색하여 VisualSVN Server 사이트로 이동한다
[SVN 설치]	
	 - Help > Eclipse Marketplace
	 - Window > Show View > Other > SVN > SVN Repository
	 - New Repository Location
	
	 - 하단창 SVN Repositories 에서 우측 클릭하여 Location Properties... 클릭
	
	 - VisualSVN Server Users > Create New User
[SVN Connertor 설치]	
	 - Preference > Team > SVN > SVN Connector 에서 Use SVN Merge view
 [프로젝트 SVN 연결]	
	 - 좌측 프로젝트 폴더에서 마우스 우측 클릭 > Team > Share Project > SVN
