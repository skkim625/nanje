- https://git-scm.com 의 우측 다운로드
- Git Bash를 반드시 포함해서 설치 : 리눅스 명령어를 사용할 수 있게 해준다
	- git --version
- Git GUI Sourcetree (선택 안하고) 설치 (Git GUI Client 는 여러 종류가 있음)
- iTerms2 를 설치하여 터미널을 분할하여 사용 

git config --global user.name
git config --global user.email

- 기본 브랜치명 변경
	- git config --global init.defaultBranch main
	
- .gitignore 에 secrets.yaml 을 입력하여 숨긴다
